Plantpedia, a silly little project by a silly little programmer

GOAL
----
CLI program that stores and displays houseplant data (name, image, date, care)
images are ascii-art representations from photos
data is stored in text and image files

BASIC FEATURES
--------------
display menu - DONE
read entry data from text/ image files - DONE
display an entry's text - DONE
display an entry's image - DONE
create an entry (text only)
edit an entry
delete an entry
find an entry by common name and latin name - DONE

ADVANCED FEATURES
-----------------
generate a static webpage
display calendar with watering dates according to watering frequency
display wikipedia info on plant
